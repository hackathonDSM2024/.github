### OPPO란?

OPPO는 사용자의 상황에 맞춰 물건 구매에 대한 결정을 도와주는 챗봇 서비스입니다. 
사용자가 물건을 구매하기 전 다양한 항목을 고려할 수 있도록
챗봇이 가이드라인에 따라 합리적인 소비인지, 충동구매인지를 평가합니다. 이를 통해 사용자가 좀 더 이성적이고 침착하게 구매 상품에 대해 고민하고, 충동 구매와 이로 인해 발생할 수 있는 문제점을 줄여보고자 개발했습니다.
